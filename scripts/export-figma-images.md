# Export Images from Figma

To export images and icons from the Figma file, follow these steps:

## Required Images:

### 1. Project Mockups (in project cards):
- KFC project mockup
- PHOTON project mockup  
- CELLEBRITE project mockup
- SEMESENCE project mockup

### 2. Tech Stack Icons:
- Design Tools icon (purple)
- Development icon (blue)
- Research & Testing icon (green)
- AI Stack icon (orange/red)

### 3. About Section:
- Profile photo

### 4. Social Media Icons:
- LinkedIn icon
- Dribbble icon
- Twitter/X icon
- Behance icon

### 5. Brand Logos:
- Honeywell logo
- Walgreens logo
- Verizon logo
- Cellebrite logo
- MedCline logo
- KFC logo

## Export Steps:

1. Open the Figma file: https://www.figma.com/design/2ecLusqLKZRgdqybxEGymD/Personal-Branding?node-id=1652-3545
2. For each image/icon:
   - Select the image/icon element
   - Right-click â†’ "Copy as PNG" or "Export"
   - Or use the Export panel (right sidebar) to export at 2x or 3x resolution
3. Save images to:
   - Project mockups: `src/assets/images/projects/`
   - Tech stack icons: `src/assets/icons/tech-stack/`
   - Profile photo: `src/assets/images/`
   - Social icons: `src/assets/icons/social/`
   - Brand logos: `public/images/brands/`

## Naming Convention:

- Project mockups: `kfc-mockup.png`, `photon-mockup.png`, `cellebrite-mockup.png`, `semesence-mockup.png`
- Tech icons: `design-tools.svg`, `development.svg`, `research.svg`, `ai-stack.svg`
- Profile: `profile-photo.jpg`
- Social icons: `linkedin.svg`, `dribbble.svg`, `twitter.svg`, `behance.svg`
- Brand logos: `honeywell.svg`, `walgreens.svg`, `verizon.svg`, `cellebrite.svg`, `medcline.svg`, `kfc.svg`
